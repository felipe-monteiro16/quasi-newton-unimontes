# Frontend - Quasi-Newton Unimontes

Interface em Vue 3 para configurar e visualizar a execucao do metodo Quasi-Newton.

## Comandos

```bash
npm install
npm run dev
```

Por padrao, o frontend tenta acessar a API em `http://localhost:8000/optimize`.
Para alterar a URL base, crie um arquivo `.env` no diretorio `frontend`:

```bash
VITE_API_BASE_URL=http://localhost:8000
```
